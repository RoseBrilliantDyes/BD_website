# Brilliant Dyes Brand Guidelines

## Colours
- Background (cream): #FFFDEC
- Primary blue: #000EAA
- Main accent (orange/coral): #F58C60
- Dark/charcoal text: #2D2D2A
- Teal accent: #7FD8D4
- Yellow accent: #FFD753

## Typography
- ALL fonts: Poppins (import from Google Fonts)
- Headings: Poppins Regular
- Body: Poppins Light and Medium
- Never use any other font

## Colours to NEVER use
- Black (#000000) backgrounds
- Dark grey backgrounds
- White (#ffffff) backgrounds — use #FFFDEC cream instead

## Logo
- Text: "brilliant dyes" lowercase
- On cream background: use blue version
- On blue background: use white version
